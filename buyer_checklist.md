# 🛒 Rumi Buyer’s Checklist (Peru)

## Step 1: Provenance Verification
- [ ] **Concession ID (INGEMMET)** → Ask the store for the mining concession number.
- [ ] **REINFO ID** → Confirm the miner is registered in REINFO.
- [ ] **Vendor RUC (SUNAT)** → Request the store’s tax ID.
- [ ] **HS Code** → Ensure the stone is classified correctly for customs.
- [ ] **VUCE COD** → If the stone is for export, ask for the Certificate of Origin code.

## Step 2: Artisan Verification (Jewelry Only)
- [ ] **Artisan RNA** → Confirm the artisan is registered in the National Artisan Registry.
- [ ] **Mount Type** → Ask how the stone is mounted (bezel, prong, filigree).
- [ ] **Artisan Technique** → Optional, but valuable for cultural storytelling.
- [ ] **Artisan Signature** → Optional digital signature/hash for authenticity.

## Step 3: Compliance Proof
- [ ] **Hedera Consensus Service (HCS)** → Ensure the compliance payload has been notarized.
- [ ] **Transaction ID** → Verify the `compliance_proof_hcs` is attached to metadata.

---

## ✅ Outcome
- **If all data is provided** → Stone/jewelry qualifies as an **artisanal product** → eligible for Rumi/Achala NFT.
- **If provenance data is missing** → Treated as a **commercial product** → excluded from Rumi.
- **If artisan RNA is missing (for jewelry)** → Cannot be minted as Achala NFT.

---

## 📌 Practical Tip
- Tourist shops and generic jewelry stores → usually **commercial products** (no provenance).  
- Export‑registered mineral companies → usually **artisanal products** (full documentation).  
- Always request **concession ID, REINFO, RUC, HS Code, VUCE COD** before purchase.
